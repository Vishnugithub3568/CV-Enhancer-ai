# CV Enhancer AI

AI-powered resume enhancement platform for students.

## Features
- Upload resume (PDF/DOCX)
- Extract resume content
- AI-based content enhancement
- Professional resume generation
- Download final resume

## Tech Stack
- Frontend: React
- Backend: FastAPI (Python)
- AI: OpenAI API
- Resume Parsing: PyPDF / python-docx
- PDF Generation: reportlab / docx

## Development Approach
This project follows an incremental development approach.
Each module (upload, parsing, AI enhancement, resume generation)
is developed and committed separately using Git version control.
